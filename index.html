Executive Program Développement Web

Formation certifiante Bac+2

Programme

La formation Executive Program Développement web permet d’acquérir les compétences essentielles et fondamentales d’un développeur web. 6 mois pour apprendre un métier, et bénéficier d’une capacité opérationnelle immédiate. Le programme s’organise en 2 temps :

Développement Front-end

Apprentissage des langages front-end (HTML, CSS, JS) Maitrise du développement responsive d’un site Conception centrée utilisateur et UX Design Création, personnalisation et administration d’un site WordPress Publication d’un site en ligne sur le serveur web

Développement Back-end

Apprentissage du langage PHP Conception et mise en œuvre d’une base de données (SQL, Merise) Développement de modules pour un site internet avec WordPress Prise en main d’un framework PHP (Symfony)
